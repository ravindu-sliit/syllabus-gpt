Syllabus-GPT: AI-Powered Educational Assistant
Project Overview: A RAG-based tool built to help SLIIT students query academic materials using natural language.

Key Features:
RAG Pipeline: Context-aware Q&A using LangChain and Google Gemini.

Local Vector Store: High-performance similarity search using FAISS.

Automated Chunking: Intelligent PDF parsing and text splitting for accuracy.

Tech Stack:
Language: Python

AI/LLM: Google Gemini 1.5 Flash, LangChain

Database: FAISS (Local Vector Store)

UI: Streamlit
